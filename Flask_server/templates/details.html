Approach Details :

Details about the system are mentioned below :

User End

1.   User will register in Webapp using email, mobile no. and password.
2.   User can upload images related to various subjects of interest.
3.   User will select the category, type of the image content and description can also be entered

Authority End


1.   Authority will be notified about the uploaded image.
2.   Authority will be provided access to the database.

Internal Processing

1.   The description given by user will be analyzed and intent will be identified in the form of keywords.
2.   For validation, the keywords from the text description will be compared with the information retrieved from the image.
3.   After the validation is complete, data will be stored in the database and the authorities concerned will be notified.
